在某些时候确实没有太适合的编辑器的,只能使用vi.掌握一些基本的vi使用还是非常
有必要的.这里记录一下自己使用vi的一些笔记<!--more-->

## 一些基本设置
### vi显示行号
临时显示行号
```
set nu
```
永久显示行号,在~/.vimrc 中写入
```shell
"显示行号
set nu

```
ps:双引号是注释

### 跳转行号
- NG → 到第 N 行 也可以使用 : N 到第N行，如 :137 到第137行）
- gg → 到第一行。（陈皓注：相当于1G，或 :1）
- G → 到最后一行
